Open Data Bot Help

This friendly chatbot, Lou, has been designed to help users interact with the City of Waterloo's Open Data Portal. Waterloo's Open Data Portal contains city-managed spatial datasets about things like parks, schools, trails, creeks, and more. Data from the portal can be used to create your own insights and applications, or just to explore authoritative data provided by the municipal government. Please feel free to chat to Lou in a humanly way to discover new datasets, visualize them on a map, learn more about their metadata, and the fields they contain. Main functionality includes:

  • Listing which datasets it has access to
  • Displaying the fields (attributes) in a given dataset
  • Displaying “more information” (metadata) for a given dataset
  • Showing geographic data on a map
  • Downloading selected datasets
  • Return the above capabilities when a user asks about functionality
